4MENIC

Conception et développement du site internet pour la réservation d’une séance de cinéma

Periode : sep - déc 2023

Lieu: EFREI

Role : Développeur, travail solo.

Descriptif : Site web destiné à la recherche des films disponibles aux cinémas, la réservation d’une seance, ou d’une salle entière, et d’un lieu virtuel où consulter les détails ou commentaires des films sortis. Les données des films sont récupérées grâce à une API REST. L’utilisateur-acheteur a la possibilité de gérer son compte personnelle avec une connexion login à partir d’une base de données MySQL et voir les séances passé et futures.

Technologie utilisée : HTML, CSS, PHP, JS, API REST

a-4MENIC

Conception et développement d'une application pour la gestion d’une salle de cinéma

Periode : janv - mars 2024

Lieu: EFREI

Role : Développeur, travail solo.

Descriptif : Application destiné aux titulaires d’une salle de cinema indépendant. Il permet de gérer les demandes des séances, l’aménagement des salles pour les éventements, la visualisation des données et statistiques liées au cinéma pour faciliter la demande de subvention. Les administrateurs pourrons se connecter au compte personnel pour attribuer chaque séance à un film.

Technologie utilisée : JAVA

pANTANI

Application interne à l'entreprise pour la gestion de fichers JSON

Periode : sept 2023 - juin 2024

Lieu: FDJ

Role : Développeur, travail en équipe.

Descriptif :
Application interne à l'entreprise destiné aux automaticiens des tests automatiques au sein de paris sportivs. Un formulaire permet de generer un fichier JSON, nécessaire pour le lancement de test par le bais d'appareils fictives en ligne.
Possibilité de creation d'un compte pour la personalisation du formulaire ou de fichers et sauvarde de données prédéfinies.
Gestion des droits des utilisateurs et modifications de formulaire.

Technologie utilisée : HTML, CSS, PHP, JS
